Building &amp; Maintenance System - System for Property Administration, Reporting, Tracking & Approval (Vue 3 ver)

## 🛠️ Tech Stack

- **Frontend Framework**: Vue 3 (Composition API)
- **Build Tool**: Vite
- **State Management**: Pinia
- **Routing**: Vue Router 4
- **HTTP Client**: Fetch API (custom wrapper)
- **Styling**: Tailwind CSS + Scoped CSS
- **Package Manager**: npm

## 📦 Installation

```bash
# Clone repository
git clone https://github.com/buildingprocess25/sparta-frontend-vue.git
cd sparta-frontend-vue

# Install dependencies
npm install

# Setup environment variables
cp .env.example .env.local
# Edit .env.local with your API_URL

# Run development server
npm run dev

# Build for production
npm run build

# Preview production build
npm run preview
